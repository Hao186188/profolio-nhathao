🚀 Võ Nhật Hào Portfolio — Personal Website
Chào mừng bạn đến với kho lưu trữ mã nguồn trang Portfolio cá nhân của Võ Nhật Hào. Đây là một trang web Single Page Application (SPA) hiện đại, được thiết kế để giới thiệu kỹ năng, dự án và hành trình phát triển trong lĩnh vực IT & Web Development.

✨ Điểm nổi bật (Features)
Thiết kế hiện đại & Responsive: Giao diện tối giản, tập trung vào trải nghiệm người dùng, tương thích hoàn hảo trên mọi thiết bị (Mobile, Tablet, Desktop).

Chế độ Dark Mode: Tích hợp tính năng chuyển đổi giao diện Sáng/Tối (Light/Dark mode) và lưu tùy chọn vào localStorage.

Hiệu ứng mượt mà (AOS): Sử dụng thư viện Animate On Scroll để tạo các hiệu ứng chuyển động chuyên nghiệp khi cuộn trang.

Preloader: Hiệu ứng chờ tải trang giúp trải nghiệm người dùng trở nên mượt mà và cao cấp hơn.

Cấu trúc rõ ràng: Phân chia các khu vực Kỹ năng, Dự án nổi bật (như job-kiengiang, MB Store) và Giới thiệu bản thân một cách khoa học.

🛠️ Công nghệ sử dụng (Tech Stack)
Ngôn ngữ chính: HTML5, CSS3 (Custom Variables), JavaScript (ES6+).

Thư viện hiệu ứng: AOS.js (Animate On Scroll).

Icon & Fonts: Font Awesome 6, Google Fonts (Inter).

Công cụ: Tailwind CSS (được tích hợp qua class), Flexbox/Grid Layout.

📁 Cấu trúc mã nguồn
index.html: Chứa toàn bộ cấu trúc trang, mã CSS (Internal) và các script điều khiển.

nhathao.jpg: Ảnh đại diện hiển thị tại phần Hero.

CV-NhatHao.pdf: File hồ sơ năng lực liên kết với nút tải CV.
